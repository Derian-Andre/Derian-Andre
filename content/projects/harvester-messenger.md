Harvester Messenger es una aplicación universal de windows, un web wrapper que contiene las redes sociales más usadas:

- Whatsapp Web
- Facebook
- Facebook Messenger
- WeChat Web
- Skype
- Telegram Web
- Hangout
- Google+
- Instagram
- Reddit
- Tweetdeck
